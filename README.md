# Reason Stock Agent 📈

這是一個自動化股票數據與財經新聞爬蟲專案，旨在協助收集與回補歷史市場數據，以供後續量化分析或 AI Agent 使用。

## 🚀 功能特色

本專案整合了多個數據源的爬蟲模組，並具備容錯機制：

1.  **台股數據 (TWStock)**: 支援回補過去 365 天的歷史股價與交易資訊。
2.  **美股數據 (USStock)**: 支援回補過去 1 年的歷史數據。
3.  **鉅亨網新聞 (Anue)**: 抓取財經新聞，支援斷點續傳機制。
4.  **Google News**: 針對特定關鍵字（預設為「台股」）搜尋過去一年的新聞聚合。
5.  **PTT Stock 版**: 抓取社群討論熱度與輿情（預設回補 60 天）。

## 📂 檔案說明

- `main.py`: **AI Agent 主程式**。這是整個專案的入口，負責協調資料收集與後續的 AI 分析流程。
- `data_collector.py`: **資料收集模組**。封裝了所有爬蟲邏輯，供主程式呼叫，負責將市場數據準備好。
- `stock_analysis.ipynb`: **資料分析 Notebook**。使用 Jupyter Lab 開啟，可直接從資料庫讀取數據並轉為 DataFrame 進行視覺化分析。
- `crawler/`: (推測) 包含各個數據源的具體爬蟲實作類別。

## 🛠️ 如何使用

### 1. 環境準備

請確保已安裝 Python 以及專案所需的套件（建議建立虛擬環境）。

### 2. 執行歷史回補

若要啟動 AI Agent (包含自動回補資料)，請執行：

```bash
python main.py
```

程式將會依序執行以下任務：
1.  台股回補 (TWStock)
2.  美股回補 (USStock)
3.  鉅亨網新聞回補
4.  Google News 回補
5.  PTT Stock 版回補

### 3. 進行資料分析

若要手動分析資料庫中的數據，請啟動 Jupyter Lab：

```bash
jupyter lab stock_analysis.ipynb
```

## ⚠️ 注意事項

- **PTT 爬蟲限制**: 由於 PTT 對請求頻率限制較嚴格，且連線較慢，回補天數預設為 **60 天**，建議穩定後再視需求調整。
- **容錯處理**: 每個爬蟲任務皆獨立運行（使用 `try-except` 包裹），單一來源失敗不會導致整個回補流程中斷。
